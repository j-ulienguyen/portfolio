Product Illustration

We were given an assignment for our MDIA 3165 - Vector Graphics class to create a coloured illustration of a mechanical object of our choice. The purpose of this assignment was to apply the skills we learned to recreate an object based off of a photograph. This illustration demonstrates the usage of basic shapes, pen tool, transformation functions, pathfinder functions, and most importantly, global swatches. All objects and any gradients created for the illustration must have a global swatch applied to them.
