Typography Posters

We were given an assignment for our MDIA 3165 - Vector Graphics class to create two Swiss-styled typographic posters with the supplied text for The Proms Classical Music Festival. Each poster uses a different type treatment: Poster 1 - Sans-serif, Poster 2 - Serif Typeface. The goal of this assignment was to understand the hierarchy of text and how a viewer will be able to easily navigate through the content of the poster. In addition, the technical requirements of this assignment was to create a modular grid to structure the layout, use of global swatches, and paragraph styles associated all text.
